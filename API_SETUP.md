# Build X - API Setup Guide

## تكوين API للنموذج المحلي

### الخطوة 1: إنشاء ملف .env

1. انسخ ملف `.env.example` إلى `.env`:
   ```bash
   cp .env.example .env
   ```

2. افتح ملف `.env` وحدث القيم:
   ```
   API_URL=https://your-model-url.trycloudflare.com
   MODEL_NAME=Build X
   ```

### الخطوة 2: تحديث رابط API

لتغيير رابط API الخاص بالنموذج المحلي:

1. افتح ملف `.env`
2. حدث قيمة `API_URL` برابط النموذج الجديد
3. احفظ الملف
4. أعد تشغيل التطبيق

### مثال على الرابط:
```
API_URL=https://premiere-geographic-telling-call.trycloudflare.com
```

### ملاحظات مهمة:

- تأكد من أن النموذج المحلي يعمل ويقبل طلبات HTTP
- الرابط يجب أن يكون متاحاً عبر الإنترنت
- التطبيق يتصل بنموذج واحد فقط ولا يمكن تغييره من الواجهة
- جميع المحادثات ستتم مع نموذج "Build X"

### استكشاف الأخطاء:

إذا واجهت مشاكل في الاتصال:
1. تأكد من صحة رابط API
2. تأكد من أن النموذج يعمل
3. تحقق من اتصال الإنترنت
4. راجع سجلات التطبيق للأخطاء
